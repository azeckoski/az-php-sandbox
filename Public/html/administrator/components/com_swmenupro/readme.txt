************* Install *************
Component Installation
----------------------
In Administrator Menu, go to Components, Install/Uninstall...
Go to the section Upload new component, browse and select the swmenupro.zip file...
Press the Upload File button.

Module Installation
-------------------
In Administrator Menu, go to Modules, Install/Uninstall...
Go to the section Upload new module, browse and select the mod_swmenupro.zip file...
Press the Upload File button.
