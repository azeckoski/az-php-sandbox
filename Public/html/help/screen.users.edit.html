User Manager - Add or Edit Users *

-- Draft Version

This screen is used to either add a new user or edit an existing one.

On the left side of the page is the User Details that need to be supplied when creating a new user, and can be edited when in that mode.

On the right side of the page is the contact details of the user. These are only present when editing a user and only if the contact has been created in the Contacts component and linked to the user. If this has been done the information will be displayed and a link at the bottom of the box provides a direct access to the Edit Contact page for that conatct.
