Menu Manager / Main Menu

This page shows a list of the Menu Items associated to the menu you have just chosen.

Toolbar Icons

New : Click this icon to create a new Menu Item

Edit : Click this icon to edit an existing Menu Item.

Publish and Unpublish : These icons have the same function as the 'Published' column icon.  Click either of these icons to publish or unpublish any items.  You may use these icons with multiple items checked.

Move : Click on this icon to open a new 'Move Menu Items' window where you can choose to move the specific Menu Item(s) -and children if desired- to another existing Menu. You may check multiple items to be moved at once.

Copy : Click on this icon to open a new 'Copy Menu Items' window where you can choose to copy the specific Menu Item(s) -and children if desired- to another existing Menu. You may check multiple items to be copied at once.

Trash : Click on this icon to send the Menu Item to the Trash Manager. You may check multiple items to be sent to the Trash Manager at once.  The menu items may be recovered through the Admin Menu: Site->Trash Manager.  Then, choose the Menu Items tab.

Cancel : Click on this icon to cancel any change made in the 'Order' column if the 'Save Order' icon has not been clicked.  You will return to the previous page.

Columns

Menu Item : This is the name of the link as it appears at your site.  Click this name so that  you can edit parameters related to this Item. A padlock icon next to it shows that it is checked-out. To check-in, click the name if it is you that has last edited the page. Then click the 'Save' or 'Cancel' icon to check back in the item.  If another user has checked out the item, please contact him/her or the administrator of your site.

Published: This shows whether the Menu Item is published or unpublished. Click  the icon to switch its state.

Reorder : This shows the order of Menu Items in the menu. Click the arrow icons to move the item up or down in the list.

Order : This shows the order of Menu Items in the menu. To change the order, type in the order number  you wish to have.  You can change the order of any two items or more this way.  Then click the 'Save Order' icon next to the Column Heading 'Order'.

NOTE: The Order and Reorder  columns show the relative positions of Parent and Children Menu Items. Moving a parent moves the children too. The order numbers are shown relatively to the Parent Item.

Access : Click the 'Access' level to toggle to another User access level for this menu item.

Itemid: This shows the 'id' for this specific Menu Item in the database.

Type: This is the type of link used for this Menu Item. If it is a 'Link – Content Item' or a ‘Link – Static Content’, you may directly edit the content by clicking the type of menu item.

CID: This shows the 'componentid' for this specific Menu Item in the database.

Other functions available above and below of the list:

Max Levels: Select the maximum number of levels deep that the menu tree is displayed on this admin page.

Filter: Type in the keyword to search for text in  the Menu Item titles.

Display #: Choose the number of menu items you wish to be displayed in the list.

NOTE: This setting will be preserved for all lists on all screens while you are in the Administator back-end. A default value for the list length is also configurable in Global Configuration.
