Menu Manager : New / Edit

Menu items have many different types depending from their role. When we create a New Menu Item we must choose his type.
The Menu Items are presented in 4 groups in this screen.  Each group lists a specific kind of menu items.  Some of them may be presented in more than one group.

Toolbar Icons

Next:  After selecting the radio button for the type of menu item you wish to create, click the "Next" icon in the Toolbar. A new window will open with the name "Add Menu Item :: The-type-of-link-chosen".

Cancel: Click on the "Cancel" Icon if you do not wish to create a Menu Item.

Help: Click on the "Help" Icon to learn more about the New Menu Item page.
