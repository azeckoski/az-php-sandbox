Trash Manager

This screen shows a list of the Content Items that is in the 'Trash bin'. Trash bin is the equivelant of the 'Recycle bin' on Windows. There are two tabs. Content Items tab, displays all the content items that have been trashed. Menu Items tab, displays all the Menu Items that have been trashed.

Toolbar

Restore: Select an item from the table and click the Restore icon to undelet it. The item will be returned to its original position, and by default, it will be unpublished.

Delete: Select an item from the table and click the delete icon, to permanently delete it.

Help: You see this help screen

Table

Title: This is the title of the trashed item
   
Section: This is the section of the trashed item.
   
Category: This is the category of the trashed item.
   
ID: This is the ID of the trashed item.
   
Menu: The menu this menu item belongs

Type: The type of menu item
