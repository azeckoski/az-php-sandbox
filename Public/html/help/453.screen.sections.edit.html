Sections Manager : New / Edit

This page lets you create or edit a Section.

Toolbar Icons

Upload: Opens a new window which lets you upload a file or an image from your computer to the images/stories folder.

Save: Saves the section and returns to Manager.

Apply: Saves the section and keep the page displayed to edit.

Cancel: Cancels the action. Any modification is lost. If opened for edit, the Section remains checked-in.

Section Details

Section Title: Type here the name of the section as you want it to display on your site.

Section Name: Type here the name of the section as you want it to appear in the database and the back-end.

Image: Select here an image to display next to the Section Description text. The dropdown lists an image choice from the images/stories folder.

Image position: Select here the position of the image relative to the description text. Left or right.

Ordering: Select here the order of the section among other sections. This will reflect in the Section Manager. New sections default to the last place. Ordering can be changed after applied or saved.

Access: This is the user access level for the category. Public, Registered or Special.

Published: Check the box to let the item show on your site.

Description:  Type here a description of the Section. This text will show —if decided so— when the section is displayed as a table, blog or other sections.

Formatting content

The formatting options interface depend on the Editor you are using.
Many editors are available as Open Source (through MamboForge) or commercial products.

Mambo comes with 2 default editors:

  • NO WYSIWYG: This editor lets you enter text and/or HTML code directly.
  • TinyMCE: This editor lets edit in WYSIWYG or HTML mode (See specific Help).

NOTE: It is advised to publish only one editor at a time in Mambot Manager and to save also that choice in Global Configuration.

NOTE: It is not advised to use the Mambot tag { mospagebreak } in this description field.

The following parameters will only show once the section is applied or saved.

Link to Menu

This will create a new menu Item in the menu you select.

Select a Menu: Highlight here one the menus defined for your site.

Select Menu Type: Select here 'SEction List', 'Section Blog' or 'Section Archive Blog' Menu Item types.

Menu Item Name: The name of the Menu as it will appear on your site.

Link to Menu: Click on this button to create the Menu Item.


Existing Menu Links

This displays a list of all Menu Items referencing this section.
