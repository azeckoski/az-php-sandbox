Public Release of Sakai Collaboration and Learning Environment

NOW AVAILABLE: Public Release Candidate 2 (Sakai 1.0 RC2)
August, 5 2004-- The latest version of the Sakai software is now available at http://cvs.sakaiproject.org/release/1.0.rc2/.

July 15, 2004

Ann Arbor, MI  --- The Sakai Project today releases its open source Collaboration and Learning Environment (CLE) software to the public.  Members of the Sakai Educational Partners Program (SEPP) have been reviewing the beta version of the software since their highly successful conference with 170 attendees in June.  This release to the public is Sakai Release Candidate 1 (RC1), and it puts the full code in the hands of any institution that wishes to begin using or understanding the Sakai software. 

“Though the Sakai Project is only six months into its 24 month plan, we are pleased to provide a public release of the Sakai software at this time,” said Joseph Hardin, Chairman of the Sakai Project Board.  “The next six weeks will further extend the software as Michigan, Indiana, MIT, and Stanford refine their local implementations of Sakai for the fall semester.”  Incremental refinements of the software will be publicly released when they are ready with the next major release of Sakai 1.5 targeted for December 2004.  Sakai 2.0 will be released in May 2005.  

The SEPP Conference made clear that Sakai partners are eager to begin building Sakai Tools now, and delivery of RC1 to the public lets institutions begin understanding Sakai’s open source software and planning their endeavors.  The Sakai APIs, Technology Portability Profile (TPP), and tools will continue to evolve with the ongoing work of the Sakai Project.  The public release also includes the Sakai Architecture Tutorial slides that were presented at the SEPP conference. (--Supporting documents and presentations can be downloaded from the Sakai Educational Partners Conference Presentations page.)

As a CLE, the Sakai software provides tools to support faculty research and service work in addition to course management.  Faculty can use Sakai’s collaboration tools to support cross-institution research projects, work with graduate students, and even committee work alongside the tools that support teaching and learning.  Users will also appreciate the pervasive WYSIWYG editor used throughout the application.  Institutions will appreciate the favorable economics in supporting one set of services (e.g., discussion forums, listservs, file repositories, etc.) to support collaboration in learning, research, and service that are delivered through a set of integrated tools.  A complete list of Sakai RC1 features is posted on the sakaiproject.org website.
