Modules: New

The User module is a custom module created when clicking the 'New' icon in the Module Manager.
You may use this module to display Content (this content is not related to Sections, Categories or Static Content Items) or an RSS Feed.

Type: mod_user

Details

Title: Type or edit the title of the module to display on the Front-end or Back-end of your site.

Show title: Choose if you want the Title to display or not on on the Front-end or Back-end of your site.

Position: Choose which position that you want this module to display in. This is determined the positions available in your templates.

Module order: Choose the order this module will appear among modules assigned to the same position in your templates.

Access Level: Choose who can view this module. Public, Registered or Special.

Published: Choose if the module contents will show on the Front-end of your Web site (if the chosen Position is available in the template).

ID: This is the ID number for the module item. This is auto-allocated by the Mambo system when the module is saved.

Description: This is a short description of the functions provided by the Module Type.

Pages / Items:

Choose which pages the module should be displayed on.

Parameters

Module Class Suffix: Add a suffix if needed to be applied to the CSS class of the module (table.moduletable) to allow individual module styling.

This information concerns RSS Feeds:

RSS URL: Enter here the URL of the RSSfeed you want to display on your site.

Feed description: Choose to show or not the description text for the whole Feed.

Feed Image: Choose to show or not the image associated with the whole feed.

Items: Enter the number of RSS Items to display. If this number is higher than the number fixed by the Syndicating site, the latter one will be used.

Item description: Type the description or intro text for individual News Items.

Word Count: Choose the the maximum number of visible words available from the description of the New Item. 0 means all text will be displayed.

Enable Cache: Select to cache or not the content of this module.

This information does not concern RSS Feeds:

Custom Output: Here you may enter any text  or code you like that you wish to display it's output  in this module anywhere on the pages of your site.
NOTE: This content may not be accessed through a Menu Item.
