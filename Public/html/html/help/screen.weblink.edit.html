Web Link Manager : New / Edit

-- Draft version

This page allow you to add or edit the details for a Web Link.
 
Name: This is the name for the Web Link.
 
Category: Select the desired category.
 
URL: Add the URL address of the Web Link.
 
Description: Add a text description for the Web Link.
 
Ordering: You can order of the category by entering the appropriate numbers.
 
Approved: Select from the list the user access level that should be able to view this Web Link.
 
Published: This is whether the Web Link is published or not.
 
When you are finished your editing, click the 'Save' icon or 'Cancel' to discard your changes.
