What's New in Version 4.5.3

Version 4.5.3 represents a milestone for the 4.5.x series of Mambo. Now Mambo is even more secure and is OSI Certified Open Source Software.

Administrator Interface

The administrator's interface now features two options:
1) Simple Interface: When selected, this option provides a less complicated interface and is primarily intended to serve the needs of new users or those with minimal content configuration needs, and
2) Advanced Interface: This is the default administration interface option and is consistent with the version 4.5.2 admin screens. This interface is designed with the needs of more advanced users and webmasters in mind, and presents the widest range of configuration flexibility.

Templates

New template samples have been added to this release.

Help System Changes

New help screens have been added and the help system has been expanded to help decrease inaccuracies in the delivery of help content.
Key References for help screens have been updated to reflect version changes.

Components

Contacts

The Hide / Display VCard option is now available as a parameter.

Content

Empty categories now display properly (when requested).
The 'email to a friend' feature has been hardened.

Menus

Link type menu items are now displayed and can be selected to influence the appearance of modules.

Bug Fixes

For details on other additions and bug fixes not listed here, please check the changelog.php file that is included with the distribution.
