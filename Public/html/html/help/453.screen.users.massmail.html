Mass Mail Users

This page allow you to send a mail a message to one or more groups of Users.

Details

Group: Make a selection from the list. Select "All User Groups" to mail the message to all users. Only one selection can be made from this list.
 
Mail to Child Groups: Check this box to automatically include all of the child groups of the selected group. For example, if you select the "Editor" group and check this box, all users in the Publisher group will also be included in the mass mail.
 
Subject: Type a subject for the message.
 
Message: Type the body of the message
 
When you are finished, click the Send Mail icon to mail the message or Cancel to abort the mail.
 
NOTE: Only a user in the Super Administrator group is permitted to use this feature.
