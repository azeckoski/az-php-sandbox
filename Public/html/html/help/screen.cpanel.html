Control Panel

-- Draft version

The control panel is the first page you see when you log into the Administrators area. You can return to it by clicking Home menu item in the top menu bar.  From the Control Panel are 9 shortcut icons to easy access the frequently used parts of the Administrator site.

Menu Manager - Click this shortcut icon access the Menu Manager.  You will be able  to  edit any of your menus. 

Front Page Manager -  Click this shortcut icon to access the Front Page Manager.  You will be able to edit which Content items appear on your Frontpage.

Static Content Manager - Click  the shortcut icon to access the Static Content Manager.  You will be able to edit these type of  items.

Section Manager - Click the shortcut icon to access the Section Manager.  You will be able to edit your Sections.

Category Manager -Click the shortcut icon to access the Category Manager. You will be able to edit your Categories.

All Content Items - Click the shortcut icon to access the All Content Items.  All Content Items  lists all of your Content Items.  You will be able to edit any of your Content Items.

Trash Manager - Click this shortcut icon access the Trash manager.  You will be able to delete permanently or to restore any items in the Trash Manager.

Global Configuration - Click this shortcut icon to access Global configuration.  You will be able to edit any of your Web site configurations.

Help - Click this shortcut icon to view the Index of any of the Help Screens.


On the right side of the screen there are four tabs:

Logged - Displays a list of users currently logged on (both in admin and the frontend). Click on a users name to go to the User Edit page for that user.

Components - Click on this shortcut tab to access any of the components and component links that you wish to edit.

Popular - Click on this shortcut tab to access the screen which lists in order, the most popular Content items on your site.  This includes the date the Content item was created and the number of  hits. Click the Content Item name to edit that item.

Latest Content - Click on this shortcut tab to access the screen which lists the most recently added Content items.  This includes the creation date and the author. Click the Content item name to edit that item.

Menu Stats - Click on this shortcut tab to access the screen which lists all menus and the number of item links in each menu.  The number of links is the total created, not the total published. Click the name of a menu to access the screen which lists the items for that menu.
